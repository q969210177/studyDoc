BKForm 基本使用 ​

vue
<BKForm :schema="schema"  @register="registerForm" />
<script lang='ts' setup>
import type { FormSchemaItem } from '@/components/BKForm/'
import  { useBKForm } from '@/components/BKForm/'
const schema:FormSchemaItem[] =[
   {
     label: '测试动态数据',
     field: 'options',
     component: 'BKSelect',
     componentProps:({formValue})=> {
       /* 这里就是根据选中的条件不同展示不同的下拉数据 */
       let options = [
           {label:"机构01",value:1},
           {label:"机构02",value:2},
           ]
       if(formValue.makeTyp===1){
           options = [
               {label:"资源01",value:1},
               {label:"资源02",value:2},
           ]
       }
       return {
           options
       }
       
     },
     defaultValue: 1,
   },
   {
     label: '生效单位',
     field: 'makeType',
     component: 'BKSelect',
     componentProps: {
       options: [
         { label: '机构', value: 0 },
         { label: '资源', value: 1 },
       ],
     },
     defaultValue: 1,
   },
   {
     label: '生效日期',
     field: 'date',
     component: 'Daterange',

     dynamicRules: () => {
       return [
         { required: false, validator: (_rule, value, callback) => {
           if (!value)
             callback()
           if (value && value[0] && value[1])
             callback()
           if (!value[0]) {
             callback(new Error('请选择开始时间'))
           }
           if (!value[1]) {
             callback(new Error('请选择结束时间'))
           }
         } },
       ]
     },
   },
]
const [registerForm, {  validate, setFieldsValue, appendSchemaByField, removeSchemaByFiled, clearValidate, getFieldsValue, getFormSchema, getFormValue, resetFields, scrollToField, validateField }] = useForm()
</script>

FormSchema 表单的渲染规则 ​

属性名类型说明
fieldstring唯一属性,提交给后台的对象中的key
valueFieldstringv-model绑定的id 默认是modelValue
componentBKFormDefines.ComponentTypeEnum需要渲染的组件,如果为nul则不会渲染
componentProps((opt: ( schema: any[]; formActionType: Partial(FormActionType); formModel: Recordable; formData: Recordable; )) => ComponentPropsType(T)) | ComponentPropsType(T)设定组件的props 可以是对象,也可以是()=>props 返回值是对象的函数
defaultValueany默认的数据
ifShowboolean | ((renderCallbackParams: RenderCallbackParams) => boolean)是否展示
colPropsPartial(ColEx)ElCol的props
classNamestringclass
formItemStyleCSSPropertiesstyle
render(renderCallbackParams: RenderCallbackParams) => VNode | VNode[] | string替代默认渲染
renderColContent(renderCallbackParams: RenderCallbackParams) => VNode | VNode[] | string替代ElCol
renderComponentContent((renderCallbackParams: RenderCallbackParams) => any) | VNode | VNode[] | string替代组件默认渲染
slotstring插槽别名
labelSlotstring | ((renderCallbackParams: RenderCallbackParams) => any)ElFormItem的label插槽
colSlotstringcolSlot的默认插槽
dynamicRules(renderCallbackParams: RenderCallbackParams) => Rule[]dynamicRules动态返回验证规则
dynamicDisabledboolean | ((renderCallbackParams: RenderCallbackParams) => boolean)动态设置是否禁用
rulesMessageJoinLabelboolean是否需要帮你补上请选择,请输入这些(没啥用 后面想想给他加点功能)
subLabelstring是否添加: 还没写 现在是统一不要冒号
changeEventstring默认是 Update:modelValue 双向绑定的事件key
changeExtraEventstring
itemPropsPartial(FormItemProps & { class: string })formItem的props
suffixstring | number | ((values: RenderCallbackParams) => string | number)后期特殊验证的插槽
labelWidthstring | numberformItem的labelWidth
showMessageboolean暂时没用 后面打算做特殊样式的校验处理
inlineMessageboolean暂时没用 后面打算做特殊样式的校验处理的
labelClassstringformItem的label 单独的class
labelstringformItem的label
requiredboolean | ((renderCallbackParams: RenderCallbackParams) => boolean)
rulesRule[]非动态验证的规则
hideRequiredAsteriskboolean是否隐藏必填校验的*号
diffValueConfigBKFormDefines.IDiffValueConfig用以配置同一组数据是否可以有相同的数据(例如两个下拉框不能选择一样的数据)

BKFormProps 基本上都是表单的props ​

属性名类型说明
schemasFormSchemaItem[]表单的渲染规则(是一个数组)
submitOnResetboolean重置按钮是否需要同时按下提交按钮
rowPropsRowPropsElRow的Props
autoSubmitOnEnterboolean按下回车 调用提交事件
resetButtonOptionsPartial(ButtonProps)重置按钮的props
submitButtonOptionsPartial(ButtonProps)提交按钮的props
showResetButtonboolean是否展示重置按钮
showSubmitButtonboolean是否展示提交按钮
resetTextstring重置按钮的text
submitTextstring提交按钮的text
transformDateFunc(date: any) => string时间转换方法(后面会支持[] 转化成两个字段的时间,这样可以不用额外的吧[]的时间拆出来 自己写 直接内部处理)
baseRowStyleCSSPropertiesrow的style
baseColPropsPartial(ColEx)col的props
fieldMapToTimeBKFormDefines.FieldMapToTime也是用来处理 一个时间组转换成两个字段的参数
autoFocusFirstItemboolean是否自动聚焦(没做)
actionColOptionsPartial(ColEx)提交,重置的col的props
mergeDynamicDataRecordable默认数据(会填充到formModel)
rulesMessageJoinLabelboolean暂时没用
autoSetPlaceHolderboolean是否需要我帮你设置 PlaceHolder(请选择,请输入)
showActionButtonGroupboolean是否展示提交和重置按钮的col
actionSpannumber不要改,我会自动计算,没啥用
actionRightstring同上
defaultExpantboolean是否默认展开
showExpantboolean是否需要展开收起的功能
defaultSliceSchemasNumnumber默认展示几个(其他的会被收起) 默认是3
actionClassstring[]
isSearchboolean是否是搜索条件
formPropsPartial(FormProps)ElForm的props

useForm的操作事件 ​

属性名类型说明
submit() => Promise(void)提交按钮的事件
setFieldsValue(values: Recordable, isClearValidate?: boolean) => Promise(void)设置formModel
resetFields(nameList?: string | string[]) => Promise(void)没参数重置所以 有参数重置部分
getFieldsValue(T = any) => T获取表单数据(仅限schemas)里面定义的
getFormValue(T = any) => T获取所有的表单数据 那怕不在schemas 里面定义的
clearValidate(name?: string | string[]) => Promise(void)清除校验
setProps(formProps: IBKFormProps) => void设置props
removeSchemaByFiled(field: string | string[]) => Promise(void)移除(推荐使用ifShow)
appendSchemaByField(schema: FormSchemaItem, prefixField: string | undefined, first?: boolean | undefined) => Promise(void)添加规则(例如在拖拉生成的表单规则)
validateField(nameList?: Arrayable(FormItemProp)) => Promise(any)校验所有或者某个字段
validate(nameList?: Arrayable(FormItemProp)) => Promise(any)校验所有或者某个字段
scrollToField(name: FormItemProp, options?: ScrollOptions) => Promise(void)滚动到某个字段
reloadParams(params: any, initVal?: any) => void没用到
setParams(params: any, newParams: any) => void没用到
getFormSchema() => FormSchemaItem[]获取所有的 schemas
